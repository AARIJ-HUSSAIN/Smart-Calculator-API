# Smart Calculator API

A simple backend API built using FastAPI that performs basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Error handling (division by zero)

## Tech Stack

* Python
* FastAPI
* Uvicorn

## How to Run

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

## API Endpoints

* `/add?a=5&b=3`
* `/subtract?a=5&b=3`
* `/multiply?a=5&b=3`
* `/divide?a=5&b=3`
